## About project

Simple blog project made of laravel framework and simple html and css

## Features

1- Create Posts<br>
2- Upload Images For Posts<br>
3- Show Any Post You Write<br>

## Technologies User

1- Laravel 12<br>
2- MySQL<br>
3- bootstrap<br>
4- HTML<br>
5- CSS<br>
